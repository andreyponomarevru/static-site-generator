# How to use

Inline:

```pug
p(class=`text_display_inline`) Lorem Ipsum is simply dummy text of the printing and typesetting industry
```

Block:

```pug
span(class=`text text_display_block`) Lorem Ipsum is simply dummy text of the printing and typesetting industry. Lorem Ipsum has been the industry's standard dummy text ever since the 1500s, when an unknown printer took a galley of type and scrambled it to make a type specimen book.
```
